# Prompt Quizzer – A JavaScript Quiz Game

This project is a simple quiz game created using JavaScript.  
The entire game runs inside the **browser console** using `prompt()` and `alert()`.

## Features
- Stores questions and answers inside an array of objects.
- Uses loops, conditionals, and functions.
- Accepts user input using `prompt()`.
- Cleans input using `toLowerCase()` and `trim()`.
- Tracks the score and displays final result.
- Fully runs in console—no HTML required.

## How to Run
1. Open your browser.
2. Press **F12** → go to **Console**.
3. Paste the full code from `quiz.js`.
4. Press Enter.
5. Play the quiz!
